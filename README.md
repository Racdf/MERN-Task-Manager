# MERN Task Manager

## Overview
Simple Task Manager built with MERN tech stack used: 

# FrontEnd
1. React.js (Component-based UI)
2. Vite (Fast development bundler)
3. Axios (API requests)
4. React Router (Navigation / Routing)
5. CSS3 (Custom modern dark UI theme)
6. Modal UI for editing tasks

# BackEnd
1. Node.js
2. Express.js (REST API framework)
3. Mongoose (ODM for MongoDB)
4. MongoDB Atlas (Cloud database)

# Deployment
Frontend: Vercel
Backend: Render (Web Service)
Database: MongoDB Atlas

## Run Backend
1. cd backend
2. npm install
3. npm run dev

## Run Frontend
1. cd frontend
2. npm install
3. npm run dev
4. Open http://localhost:5173

## APIs (Tested with Thunder Client extension)
- POST /api/tasks
- GET /api/tasks
- GET /api/tasks/:id
- PUT /api/tasks/:id
- DELETE /api/tasks/:id
