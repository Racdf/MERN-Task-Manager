# MERN Task Manager

## Overview
Simple Task Manager built with MERN stack: React, Node/Express, MongoDB, Mongoose.

## Run Backend
1. cd backend
2. npm install
3. npm run dev

## Run Frontend
1. cd frontend
2. npm install
3. npm run dev
4. Open http://localhost:5173

## APIs
- POST /api/tasks
- GET /api/tasks
- GET /api/tasks/:id
- PUT /api/tasks/:id
- DELETE /api/tasks/:id
